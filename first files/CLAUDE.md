## Project Overview

This is a **Google Ads campaign management project** (not a software codebase) for MCP Muebles, a custom kitchen furniture manufacturer in Santa Fe, Argentina. The project contains marketing strategy documents, campaign performance screenshots, and optimization plans.

**Client:** MCP Muebles (Marcelo) — www.mcpmuebles.com
**Objective:** Generate qualified WhatsApp leads for custom kitchens (cocinas a medida)
**Location:** Santa Fe, Argentina

## Repository Contents

- `google-ads-plan.md` — Main strategy document with diagnostics, action items, campaign restructure proposal, keywords, ad copy, budget, and implementation checklist
- `img_00.png` to `img_23.png` — Google Ads interface screenshots and landing page captures documenting current campaign state

## Key Context

- 3 active campaigns: main search (85% of traffic), presupuestador (best CTR), mesadas (smallest)
- All conversions funnel to WhatsApp (no web forms)
- Client has multiple landing page designs (dark/elegant, blue/modern product page, green/bold presupuestador)
- Currency is ARS (Argentine Pesos)
- All ad copy and landing pages are in Spanish (Argentine dialect — voseo)

## Working With This Project

- When editing `google-ads-plan.md`, maintain the existing structure and Spanish language
- Campaign data is point-in-time (Feb-Apr 2026); always note dates when adding new data
- Use Argentine Spanish conventions (vos instead of tú, pesos argentinos)
